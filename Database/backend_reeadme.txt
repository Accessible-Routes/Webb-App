To run the backEnd server:
1. install PostgreSQL
2. 'psql -U postgres' to get into psql command
3. 'node index.js' to connect to the database
4. in psql command, 
    '\c rim': Expected output: 'You are now connected to database "rim" as user "postgres".'
    '\d' to show database tables 
    '\d+ tableName' to show table details
